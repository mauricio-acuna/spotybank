# LEEME PRIMERO — Guía rápida del proyecto

## ¿Por dónde empezar?

1. Lee `00_GESTION_PROYECTO\statusObraLiteraria.md` para ver el estado real del ciclo actual.
2. Consulta `09_REVISIONES\CIERRE_TECNICO_CAPITULOS_v0.2-editorial.md` para revisar el cierre capitulo por capitulo.
3. Consulta `09_REVISIONES\VALIDACION_INTEGRAL_CIERRE_TECNICO_v0.2-editorial.md` para revisar la consistencia global antes de publicacion.
4. Usa `02_PLANIFICACION\INDICE_PROVISIONAL.md` como mapa de lectura actualizado.
5. Revisa `10_PUBLICACION\CHECKLIST_SALIDA_v0.2-editorial.md` solo cuando se decida ordenar release, formato de entrega o tag.

## Convención de nombres de fichero

- Capitulos cerrados: `CAP_01_BORRADOR.md` mantiene nombre historico, pero el contenido puede estar en `CERRADO_TECNICO`.
- Revisiones globales: `09_REVISIONES\*.md`
- Publicacion: `10_PUBLICACION\*.md`
- Figuras     : `FIG_01_descripcion.png`
- Tablas      : `TAB_01_descripcion.xlsx`
